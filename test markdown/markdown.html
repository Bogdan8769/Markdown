Markdown este un limbaj de marcare ușor care îți permite să formatezi textul folosind o sintaxă simplă și ușor de citit. Creat în anul 2004 de John Gruber și Aaron Swartz, scopul său principal este de a transforma textul simplu într-un format vizual plăcut (de exemplu, convertit în HTML) fără a fi nevoie de editori complecși de text sau de cunoștințe de programare. [1, 2, 3] Fișierele scrise în acest format folosesc de obicei extensia .md. [4, 5]

De ce este atât de popular?

  • Simplitate: Se învață în doar câteva minute.
  • Portabilitate: Textul rămâne lizibil chiar dacă este deschis într-un editor de text simplu.
  • Compatibilitate: Este folosit pe platforme mari precum GitHub, Reddit, Discord, Microsoft Teams sau Genius. [1, 2, 3, 6, 7]
