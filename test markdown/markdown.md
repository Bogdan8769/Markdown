Markdown este un limbaj de marcare ușor care îți permite să formatezi textul folosind o sintaxă simplă și ușor de citit. Creat în anul 2004 de [John Gruber și Aaron Swartz](https://en.wikipedia.org/wiki/Markdown), scopul său principal este de a transforma textul simplu într-un format vizual plăcut (de exemplu, convertit în [HTML](https://docs.fileformat.com/ro/word-processing/md/)) fără a fi nevoie de editori complecși de text sau de cunoștințe de programare. [1, 2, 3] 
Fișierele scrise în acest format folosesc de obicei extensia .md. [4, 5] 
## De ce este atât de popular?

* Simplitate: Se învață în doar câteva minute.
* Portabilitate: Textul rămâne lizibil chiar dacă este deschis într-un editor de text simplu.
* Compatibilitate: Este folosit pe platforme mari precum GitHub, Reddit, Discord, [Microsoft Teams](https://support.microsoft.com/ro-RO/teams/chat/use-markdown-formatting-in-microsoft-teams) sau [Genius](https://genius.com/Genius-romania-ghid-de-formatare-si-markdown-annotated). [1, 2, 3, 6, 7] 

## Exemple de sintaxă Markdown

| Rezultat dorit | Ce trebuie să scrii în Markdown |
|---|---|
| Titlu mare | # Titlu |
| Text îngroșat (Bold) | **text îngroșat** |
| Text cursiv (Italic) | *text cursiv* |
| Link către Google | [Link către Google](https://google.com) |
| Listă cu puncte | * Element 1 * Element 2 |
