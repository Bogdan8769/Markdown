Proiectul Meu de Conversie

Bine ai venit pe noua mea pagină web! Acest fișier a fost convertit automat din Markdown direct în HTML folosind un script Python scris de mine. Cât de tare e asta?

Ce am învățat să fac astăzi:

  • Să instalez o bibliotecă externă folosind pip.
  • Să citesc text dintr-un fișier și să creez unul nou.
  • Să scriu funcții pentru a refolosi codul mai ușor.

Limbaje și unelte folosite:

  1. Python (pentru logică)
  2. Markdown (pentru scrierea textului)
  3. HTML (pentru afișarea în browser)

"Orice programator bun a început prin a scrie un cod simplu care funcționează."

Dacă vrei să afli mai multe despre bibliotecă, poți căuta detalii despre Python Markdown.
